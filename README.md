# PlottingTonalSpace
Plotting library for note distributions in different representations of tonal space in Python.

## Dependencies
- Pandas
- Matplotlib
- Numpy

## Tonal Space Representations
- Circle of fifths
- Circle of chromatics
- Tonnetz

### Coming soon
